# Radiance Cascades
This is a repo with a bunch of LegitLS scripts for exploring Radiance Cascades (paper: https://github.com/Raikiri/RadianceCascadesPaper).

Any of these scripts can be previwed in our web-based multipass shader editor LegitSL, for example: https://radiance-cascades.github.io/LegitScriptEditor/?gh=Raikiri/LegitCascades/Scaling.ls

LegitSL source: https://github.com/radiance-cascades/LegitScriptEditor
LegitScript source (the language that LegitSL runs): https://github.com/Raikiri/LegitScript
